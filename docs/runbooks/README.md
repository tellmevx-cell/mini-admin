# 运行手册目录

这个目录用于保存平台能力的操作手册、验收手册和上线检查清单。

当前收口重点是工作流和消息中心，不包含具体业务模块，也不包含外部业务平台接入。

## 手册列表

- [工作流与消息中心操作手册](workflow-message-center-operations-manual.md)
- [工作流与消息中心验收手册](workflow-message-center-acceptance-checklist.md)

## 使用建议

- 日常培训、交接和问题排查时，优先看操作手册。
- 上线前、合并大改后、接入新业务前，按验收手册跑一遍。
- 如果后续新增业务模块，业务自己的验收点应单独建文档，不写进这两份平台手册。
