# 通知公告需求文档

> 回补整理。

## 背景

通知公告是后台系统常见内容模块，用于发布系统维护、版本更新、业务通知等信息。当前阶段先完成后台维护能力，为后续首页通知、消息中心打基础。

## 目标

- 支持通知公告分页查询。
- 支持通知公告新增、编辑、删除。
- 支持标题、类型、内容、状态、发布时间等字段。
- 接入菜单和按钮权限。

## 功能范围

- 通知公告管理页面。
- 通知公告 CRUD 接口。
- 权限码控制。

## 不做范围

- 不做用户已读未读。
- 不做站内信推送。
- 不做富文本上传图片。

## 验收标准

- [x] 能维护通知公告。
- [x] 关闭通知公告菜单权限后前端不展示菜单。
- [x] 接口按权限码保护。
