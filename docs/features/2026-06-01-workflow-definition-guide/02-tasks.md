# 工作流定义示例引导任务执行文档

## 任务拆分

1. 增加流程定义引导状态。
   - `definitionGuideOpen`
   - `definitionGuideStep`
   - `definitionGuideSavedDefinitionId`

2. 增加请假流程草稿生成逻辑。
   - 自动生成唯一流程编码。
   - 自动填入流程名称、表单名称、说明。
   - 自动创建主管审批、部门负责人审批、条件节点、抄送节点和结束节点。
   - 自动配置 `days > 3` 长假分支和默认分支。

3. 增加分步引导弹窗。
   - 左侧步骤列表。
   - 右侧当前步骤说明。
   - 支持上一步、下一步、保存并继续、跳转发起测试。

4. 增强页面可见性。
   - 顶部增加“示例引导”入口。
   - 流程定义工具栏增加“示例引导”和“生成请假示例”。
   - 当前引导步骤关联左侧列表、画布、右侧属性面板高亮。

5. 验证。
   - 执行前端生产构建。
   - 确认后端健康检查。
   - 确认前端页面返回 200。
