# 工作流定义示例引导总结文档

## 已完成

- 在审批中心顶部和流程定义工具栏增加“示例引导”入口。
- 新增“创建工作流示例引导”弹窗。
- 支持一步一步完成请假工作流示例：
  - 第 1 步：生成请假流程草稿。
  - 第 2 步：确认流程基础信息。
  - 第 3 步：认识画布节点。
  - 第 4 步：配置审批人和条件分支。
  - 第 5 步：保存流程定义。
  - 第 6 步：跳转发起审批并填入示例请求。
- 示例流程包含：
  - 开始。
  - 直属主管审批。
  - 请假天数判断。
  - 部门负责人审批。
  - 抄送人事。
  - 结束。
- 条件分支示例：
  - `days > 3`：进入部门负责人审批。
  - 默认分支：3 天以内直接结束。
- 保存仍复用原有校验，审批人缺失或条件分支缺失时会阻止保存并提示。
