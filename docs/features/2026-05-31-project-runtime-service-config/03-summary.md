# 项目运行管理服务配置总结

## 本次完成

- 在项目运行管理页面支持服务新增、配置和删除。
- 服务配置覆盖基础信息、运行配置和打包配置。
- 服务类型提供 `.NET API`、`.NET 通用`、`Vue / Vben`、`React`、`uniapp`、`前端通用`、`自定义` 模板。
- 保存时复用项目更新接口，前端会完整回写当前项目、工作区和服务配置，避免只保存单个服务导致其他字段丢失。

## 使用方式

1. 进入 `系统监控 / 项目运行管理`。
2. 选择项目和工作区。
3. 点击 `新增服务` 创建服务，或点击服务行右侧 `配置` 修改已有服务。
4. 选择服务类型可快速填充常见启动命令、打包命令和产物路径。
5. 保存后可继续使用启动、停止、重启、打包、打开产物等动作。
