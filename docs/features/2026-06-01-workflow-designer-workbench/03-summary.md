# 工作流设计器工作台总结文档

## 已完成

- 对流程定义页面进行了布局重构：
  - 左侧流程库。
  - 中间大画布。
  - 右侧属性面板。
- 移除了流程定义主编辑流程对弹窗的依赖。
- 支持在画布中新增多种节点：
  - 审批节点。
  - 条件节点。
  - 抄送节点。
  - 结束节点。
- 为 Vue Flow 增加了自定义节点外观，让节点类型更清晰。
- 保存逻辑继续兼容现有后端：
  - 审批节点进入后端可执行节点列表。
  - 自定义设计节点进入 `DesignerJson`。
- 增强保存前校验：
  - 未填写流程名称或编码时阻止保存。
  - 审批节点未填写节点名称、审批用户或审批角色时阻止保存。
  - 自动选中配置不完整的节点，并用弹窗和右侧提示说明缺失项。
- 明确了当前后端是自研轻量 .NET workflow 引擎。

## 验证结果

- `npx impeccable --json ...workflow/center/index.vue`：未检出自动化设计反模式。
- `pnpm run build:antd`：通过。
- 保存前校验增强后再次执行 `pnpm run build:antd`：通过。
