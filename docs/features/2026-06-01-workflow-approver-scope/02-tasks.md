# 工作流审批人范围修复任务

## 后端

- 增加工作流专用审批用户接口：`GET /workflow/approver/users`。
- 增加工作流专用审批角色接口：`GET /workflow/approver/roles`。
- 两个接口都按照当前平台/租户范围过滤数据。
- 角色接口只返回启用且当前范围内至少有启用用户的角色。
- 流程发起和审批流转创建待办时，传入当前操作人上下文。
- 找不到审批人时，根据用户、角色、禁用、租户不匹配等场景生成明确错误。

## 前端

- 流程定义页面不再复用系统用户列表和系统角色列表。
- 审批人下拉改用工作流专用审批人接口。
- 审批角色下拉显示当前可审批人数。
- 当前范围没有可用用户或角色时，在节点属性区域给出提示。
