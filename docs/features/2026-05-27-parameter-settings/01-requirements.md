# 参数设置需求文档

> 回补整理。

## 背景

系统参数用于维护运行时可配置项，例如站点名称、开关类配置、业务阈值等。它适合保存不需要重新发版即可调整的轻量配置。

## 目标

- 支持参数分页查询。
- 支持参数新增、编辑、删除。
- 参数包含键、名称、值、分组、备注、状态。
- 后续可以扩展为业务配置读取来源。

## 功能范围

- 参数设置页面。
- 参数 CRUD 接口。
- 权限码控制查询、新增、编辑、删除。

## 验收标准

- [x] 能查询系统参数。
- [x] 能新增、编辑、删除参数。
- [x] 参数 key 可用于后续业务读取。
- [x] 前后端权限控制正常。
