# 岗位管理需求文档

> 回补整理。

## 背景

岗位用于描述用户在组织内的职务，例如管理员、开发工程师、测试工程师。它不直接决定权限，但可以辅助用户筛选和组织信息展示。

## 目标

- 支持岗位分页查询。
- 支持岗位新增、编辑、删除。
- 用户可以绑定岗位。
- 用户列表展示岗位名称。
- 用户列表支持岗位筛选。

## 功能范围

- 岗位管理页面。
- 岗位 CRUD 接口。
- 用户新增/编辑选择岗位。
- 用户列表按岗位过滤。

## 验收标准

- [x] 能维护岗位。
- [x] 用户可以绑定岗位。
- [x] 用户列表展示岗位名称。
- [x] 用户列表可按岗位筛选。
