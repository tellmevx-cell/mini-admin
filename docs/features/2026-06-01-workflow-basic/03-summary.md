# 工作流一期总结文档

## 已完成

- 新增工作流领域模型：
  - 流程定义。
  - 审批节点。
  - 流程实例。
  - 待办任务。
  - 流转记录。
- 新增后端服务与接口：
  - 流程定义列表、选项、新增、编辑、删除。
  - 流程实例列表、详情、发起。
  - 我的待办、我的已办。
  - 同意、驳回、撤回。
- 新增 MySQL 表兜底初始化和 EF InMemory 支持。
- 后端工作流执行升级为 .NET 图执行：
  - 保存 `DesignerJson`。
  - 审批节点保存 `DesignerNodeId`。
  - 发起时根据开始节点连线找到第一个审批节点。
  - 审批通过后根据当前节点连线寻找下一审批节点或结束节点。
- 前端接入 `@vue-flow/core`，流程定义改成节点式可拖拽画布：
  - 左侧节点面板。
  - 中间流程画布。
  - 右侧节点属性。
  - 支持拖拽节点、重新连线、保存布局。
- 新增菜单权限种子：
  - 工作流。
  - 审批中心。
  - 查询、定义管理、发起审批、审批处理权限。
- 默认租户管理员和默认套餐会补齐工作流菜单权限。
- 新增前端审批中心页面：
  - 我的待办。
  - 发起审批。
  - 流程实例。
  - 流程定义。
  - 流程详情弹窗。
- 新增后端测试覆盖“发起审批 -> 生成待办 -> 审批通过”的核心闭环。
