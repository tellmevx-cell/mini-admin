# 密码修改与重置完工总结

## 完成内容

- 当前用户可以修改自己的密码。
- 管理员可以重置用户密码。
- 重置密码按钮接入 RBAC 权限。
- 密码变更后旧 token 失效。
- 密码错误、确认不一致、策略不满足会返回明确提示。

## 关键实现

- `PasswordService` 负责密码哈希和校验。
- `UserAppService` 负责修改、重置和业务校验。
- `PasswordOperationResult` 统一表达密码操作结果。
- 用户安全版本变化后旧 token 不再通过校验。

## 后续建议

- 增加首次登录强制修改密码。
- 增加密码过期策略。
- 增加密码复杂度配置项。
