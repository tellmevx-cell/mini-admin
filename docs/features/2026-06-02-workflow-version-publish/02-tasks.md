## 数据约定

- `Version`：同一流程编码下从 1 开始递增。
- `PublishStatus`：
  - `Draft`
  - `Published`
  - `Archived`
- `PublishedAt`：发布时间。

## 数据流

```mermaid
sequenceDiagram
    participant A as 管理员
    participant D as 流程定义
    participant I as 流程实例

    A->>D: 创建草稿 v1
    A->>D: 发布 v1
    I->>D: 使用 Published v1 发起审批
    A->>D: 尝试编辑 v1
    D-->>A: 拒绝，已有实例
    A->>D: 复制新版本 v2 草稿
    A->>D: 发布 v2
    D->>D: 归档 v1
```
