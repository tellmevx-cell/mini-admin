# Code Generator Workflow Guide Requirements

## 背景

启用“审批接入”生成业务模块后，前端开发服务可能因为新增 Vue 文件触发热更新或页面刷新。若浏览器请求在后端写完文件后被取消，生成历史可能没有落库，用户会看到表已经变成“已映射”，但历史记录里没有本次生成。

同时，审批接入生成的是业务模块与工作流的代码接口，不等于自动产生审批待办。用户还需要绑定业务类型、发布流程、提交业务单据，审批人才会在工作流中心看到待办。

## 目标

- 代码生成一旦开始写入文件，就不能因为前端刷新取消而丢失生成历史。
- 生成成功后历史列表应自动聚焦本次生成模块。
- 启用审批接入生成成功后，应提示后续步骤：重启后端、绑定业务类型、提交审批、进入待办审批。
- 保持原有预览、冲突检测、回滚能力不变。

## 非目标

- 本阶段不自动重启后端。
- 本阶段不自动创建流程定义。
- 本阶段不自动把生成模块绑定到某个流程。
