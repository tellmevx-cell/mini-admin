# UI 审查与轻量优化执行

## 自动审查

执行：

```powershell
npx impeccable --json --fast C:\monica\code\mini-admin\frontend\vue-vben-admin\apps\web-antd\src\views
```

结果为空数组，说明未发现自动规则可识别的明显设计反模式。

## 人工审查结论

- 系统监控摘要卡片的说明文字使用单行省略，依赖异常信息较长时不利于排查。
- 代码生成器详情抽屉、配置区和步骤区在窄屏下仍保持多列，容易产生挤压。
- 项目运行管理的服务配置弹窗字段较多，缺少明确的弹窗体滚动边界。

## 调整内容

- 系统监控：摘要说明改为最多两行展示，保留稳定高度，并允许长文本换行。
- 代码生成器：补充移动端响应式规则，让配置头、底部动作、详情指标和步骤标题在窄屏下纵向排列。
- 项目运行管理：为服务配置弹窗设置独立包裹类和最大可滚动高度，避免长表单超出视窗。

## 不调整内容

- 不改变功能入口。
- 不改变接口请求。
- 不改变表格字段、按钮权限和业务流程。
