# UI 审查与轻量优化需求

## 背景

本次针对后台现有页面做设计审查，不新增业务功能，不调整接口，只修正影响企业后台使用体验的展示细节。

## 范围

- 扫描 `frontend/vue-vben-admin/apps/web-antd/src/views` 下所有 Vue 页面。
- 重点人工审查自定义程度较高的页面：
  - 系统监控
  - 代码生成器
  - 项目运行管理

## 优化目标

- 保持当前系统安静、清晰、偏工具型的企业后台风格。
- 减少窄屏或信息较长时的布局压迫感。
- 保留功能入口和业务行为不变。
- 优先修复能明确改善可读性、滚动边界和响应式稳定性的样式问题。
