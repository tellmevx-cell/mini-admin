## 实现注意

- 通知表按 `UserId + SourceType + SourceId` 建唯一索引，保证同源通知幂等。
- 通知 API 不挂菜单权限，只要求登录，因为它属于当前用户基础能力。
- 告警通知标题保留告警等级，内容保留告警正文，链接指向 `/system/alert`。
- 站内通知不复用 `mini_notices`，避免把个人已读状态和公告管理耦合。
